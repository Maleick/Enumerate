Enumerate
======

Enumerate all the external things!

## Install

```shell
sudo git clone https://github.com/Maleick/Enumerate.git /opt/Enumerate
cd /opt/Enumerate
sudo ./install.sh
```

## Usage

Enumerate requires:
- root privileges
- list of IP addresses

```shell
sudo enumerate iplist.txt
```
